02 · Productions

Productions réalisées

Quatre livrables principaux ont été développés dans le cadre de cette réalisation professionnelle.

01

Base de données relationnelle SQL

Schéma comprenant les tables Véhicules, Marques, Conducteurs, Affectations et Maintenances. Clés primaires et étrangères pour l'intégrité référentielle. Modèle normalisé MCD → MLD sous MySQL Workbench.

02

Interface Homme-Machine (IHM) — Windows Forms / WPF

Formulaires avec validation par regex (plaques, dates CT), composants DataGridView pour l'affichage dynamique et filtré des données, tableau de bord synthétique avec alertes.

03

Logique métier CRUD en C#

Implémentation complète des opérations Create / Read / Update / Delete sur l'ensemble des entités, avec couche d'accès aux données (DAL) isolée via ADO.NET / Entity Framework.

04

Documentation technique complète

Commentaires XML dans le code C#, README d'installation, dossier docs/ comprenant MCD/MLD, captures UI et dictionnaire des données.

03 · Architecture

Architecture 3-Tier

L'application suit un modèle en couches séparant clairement l'interface graphique, la logique métier et l'accès aux données.

COUCHE 1

UI Layer

Windows Forms / WPF — Formulaires, DataGridView, tableau de bord, validations.

COUCHE 2

BLL — Business Logic

Règles métier, validation (regex plaques, dates), alertes CT et garanties.

COUCHE 3

DAL — Data Access

ADO.NET / Entity Framework — Requêtes SQL paramétrées, connexion MySQL localhost.
