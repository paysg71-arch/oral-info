02 · Productions

Productions réalisées

L'ensemble de l'architecture d'un site de vente en ligne a été développé, respectant les normes actuelles.

01

Interface Web dynamique et responsive

Développement de l'intégration front-end en HTML5, CSS3 et JavaScript, avec une utilisation de Bootstrap 5 pour garantir un design adaptatif sur mobile et desktop.

02

Logique métier en PHP 8

Création d'un système robuste exploitant les sessions PHP pour l'authentification client et la persistance du panier, garantissant des transactions sécurisées.

03

Base de données relationnelle

Modélisation et implémentation de la DB MySQL structurant les utilisateurs, catégories, produits, et commandes pour garantir l'intégrité des informations e-commerce.

04

Modélisation & Documentation technique

Élaboration des diagrammes UML (cas d'utilisation, séquence), modèles Merise (MCD/MLD) ainsi que les instructions de déploiement et d'installation (README).

03 · Architecture

Architecture Client-Serveur

L'application web repose sur une architecture robuste séparant la vue utilisateur, la logique de serveur et la persistance des données.

FRONT-END

Interface Utilisateur

HTML5, CSS3, JavaScript et Bootstrap 5 pour un affichage interactif, dynamique et responsive.

BACK-END

Logique Applicative

Scripts PHP 8 pour le routage, la session utilisateur et le traitement des requêtes e-commerce.

DATABASE

Persistance des Données

Base de données MySQL locale hébergée sur environnement WAMP / XAMPP.
